Intermediary BackEnd - Faculdade Digital

Visão Geral:

O projeto "Intermediary BackEnd" simula uma faculdade digital, com funcionalidades de cadastro de professores, usuários, avaliações de cada professor e cursos com seus respectivos professores já cadastrados.

Tecnologias Utilizadas:

  • BackEnd em Node.js
  • FrontEnd em Angular
  • Banco de Dados Postgres

Objetivo:

O objetivo deste projeto é integrar uma aplicação BackEnd com uma aplicação FrontEnd, além de entender conceitos como versionamento de código, framework Node.js com seus módulos internos e externos.

Ferramentas Utilizadas:

  • IDE: Visual Studio Code
  • Navegador: Google Chrome
  • Versionamento: Git
  • Gerenciamento de Branches: Sourcetree
